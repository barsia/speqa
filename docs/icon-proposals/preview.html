MarQA 16x16 File Icon Proposals

3 approaches x 3 status variants. Each shown at actual 16px and at 5x zoom on light and dark backgrounds.

Approach A: Scalloped-Edge Stamp Outline

Uses a continuous scalloped/wavy path for the stamp edge (no individual perforation circles). Inner element: pencil (draft), checkmark (ready), strikethrough (deprecated).

Approach B: Perforation Dots + Status Badge Overlay

Keeps individual perforation dots (like current icons) but adds content lines inside and a small colored badge in the bottom-right corner with a white glyph (pencil, checkmark, X). Badge sits outside the rotation for crispness. Follows the IntelliJ convention of status overlays.

Uses the scalloped edge from Approach A, but instead of status glyphs, the inner area shows wavy postal cancellation lines. Dashed = draft (not yet cancelled), solid = ready (approved/cancelled), faded + strikethrough = deprecated. This is the most "postal" of the three.

Design Notes

  • Approach A — Cleanest. Single-path scalloped edge is crisp at 16px. Status is conveyed by both color AND an inner glyph (pencil/check/strike). Most readable at actual size. Recommended if you want maximum clarity.
  • Approach B — Most similar to current icons (keeps perforation dots). Adds content lines inside for visual weight, plus a badge overlay in the IntelliJ style. The badge makes status immediately obvious even without parsing color. Best if you want to follow IntelliJ's overlay convention.
  • Approach C — Most thematically "postal". The wavy cancellation lines are a recognizable postal motif. Status is conveyed through line style (dashed/solid/faded). Most creative but potentially hardest to read at actual 16px. Best if brand identity matters more than instant status recognition.
